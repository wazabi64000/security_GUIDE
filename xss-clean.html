Protection contre XSS avec xss-clean

Description

xss-clean est un middleware qui nettoie les entrées utilisateur pour supprimer les scripts malveillants (protection contre les attaques XSS).

Cas d'utilisation

Formulaires de contact, commentaires, champs de profil utilisateur ou tout point d’entrée manipulé par l’utilisateur.

Pourquoi l'utiliser

Les attaques XSS permettent d’injecter du code JavaScript dans une application via des entrées utilisateur. Cela peut entraîner le vol de cookies, le détournement de sessions ou d'autres actions malicieuses.

Installation

npm install xss-clean

Exemple

import express from 'express';
import xssClean from 'xss-clean';

const app = express();

app.use(express.json());
app.use(xssClean());

app.post('/message', (req, res) => {
  const message = req.body.message;
  res.send(`Message reçu : ${message}`);
});

app.listen(3000);

Explication

- xss-clean() inspecte le corps de la requête, les paramètres et les headers.
- Il supprime automatiquement les balises ou scripts potentiellement dangereux.
- Protège les routes sensibles des scripts injectés par l'utilisateur.
